#  ⚠️  DO NOT use `pip install opencv-python` in Termux.
#  It will try to compile OpenCV from C++ source and hang.
#
#  Instead, install via pkg (pre-compiled binaries):
#
#    pkg install python-numpy python-opencv
#
#  Then this app has NO pip dependencies.
#  If you still want to use pip for numpy only:
#
#    pip install numpy
